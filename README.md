# Smart Motor Health Monitoring System ⚙️📊

An IoT-based Smart Motor Health Monitoring System developed as a 4th Semester Capstone Group Project.  
The project focuses on monitoring motor parameters in real-time to improve efficiency, detect abnormalities, and reduce unexpected failures through predictive maintenance techniques.

---

## 🚀 Features

- Real-time motor monitoring
- Sensor-based data collection
- Fault and abnormality detection
- Predictive maintenance support
- Live parameter visualization
- IoT integration for smart monitoring

---

## 🛠️ Technologies Used

- Arduino / ESP32
- Embedded C / Arduino IDE
- IoT Sensors
- Serial Communication
- Motor Monitoring Modules

---

## 📌 Project Objective

The main objective of this project is to continuously monitor motor health parameters and identify potential issues before failure occurs. This helps in:

- Improving motor efficiency
- Reducing maintenance cost
- Preventing sudden breakdowns
- Increasing equipment lifespan

---

## ⚙️ Hardware Components

- Microcontroller (Arduino/ESP32)
- Motor
- Sensors
- Power Supply Module
- Connecting Wires
- Breadboard / PCB
